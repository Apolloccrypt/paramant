Ghost Pipe Protocol

Post-quantum stateless data transport. Not a database. Not a message queue. A zero-knowledge RAM relay that forgets everything.

ML-KEM-768AES-256-GCM BURN-ON-READ5MB FIXED BLOCKS EU / HETZNERNO ACCOUNTS OPEN SOURCE
What Ghost Pipe doesYou push an encrypted blob. Someone pulls it once. The relay immediately destroys its copy. After 300 seconds, any unclaimed blob is wiped from RAM. There is no disk. There is no database. There is no log of content.

Quickstart

Send your first encrypted blob in under 3 minutes.

1. Generate a keypair

# Python — no dependencies beyond stdlib
from paramant import GhostPipe

gp = GhostPipe()  # generates ephemeral ECDH P-256 keypair
print(gp.public_key_hex)  # share this with the sender

2. Sender encrypts and pushes

# Sender side — encrypt locally, push to relay
gp_sender = GhostPipe()
result = gp_sender.send(
    data=open("secret.json", "rb").read(),
    recipient_pubkey="04a3f8c2d1..."  # from step 1
)
print(result["hash"])   # send this to the recipient
print(result["ttl_ms"]) # 300000 — 5 minutes

3. Recipient pulls and burns

# Receiver side — pull + burn-on-read + decrypt
data = gp.receive(hash="a3f8c2d1e7b4...")
# relay copy is now destroyed
# data is bytes in local RAM only
That's it. No accounts. No TLS certificates to manage. No server infrastructure on your end. The relay handles routing via SHA-256(pubkey).

Fixed-Block Padding

All blobs must be exactly 5,242,880 bytes (5MB). This prevents traffic analysis — the relay cannot distinguish a 10-byte message from a 4MB file.

// Block layout (5MB total)
[0..8]       = data_length as u64 big-endian
[8..8+n]     = encrypted_data (n bytes)
[8+n..5MB]   = cryptographically random noise

// Python implementation
import struct, secrets

def pad(data: bytes) -> bytes:
    BLOCK = 5 * 1024 * 1024
    block = bytearray(BLOCK)
    struct.pack_into(">Q", block, 0, len(data))
    block[8:8+len(data)] = data
    block[8+len(data):] = secrets.token_bytes(BLOCK - 8 - len(data))
    return bytes(block)

def unpad(block: bytes) -> bytes:
    n = struct.unpack_from(">Q", block)[0]
    return block[8:8+n]

Crypto Specification

Key exchange

// 1. Recipient generates keypair
recipient_kp = ECDH_P256.generate()
routing_hash = SHA256(recipient_kp.public_key)[0..32]  // 64 hex chars

// 2. Sender encrypts
ephemeral_kp  = ECDH_P256.generate()           // fresh per message
shared_bits   = ECDH(ephemeral_kp.priv, recipient_kp.pub)
aes_key       = HKDF_SHA256(shared_bits, salt=ephemeral_pub, info="paramant-ghost-pipe-v2")
iv            = random_bytes(12)
ciphertext    = AES_256_GCM.encrypt(aes_key, iv, plaintext)

// 3. Wire format
encrypted = ephemeral_pub(65B) || iv(12B) || ciphertext || gcm_tag(16B)
padded    = pad(encrypted)  // → exactly 5MB
b64       = base64(padded)  // → POST body
Note on ML-KEM-768The routing and encryption currently use ECDH P-256 (quantum-vulnerable). ML-KEM-768 encapsulation is available via paramant-core and will be the default in v2.1. For "harvest now, decrypt later" protection, use the Rust SDK which integrates the full hybrid KEM stack.

Burn-on-Read

This is not a feature — it's a protocol invariant. The relay implementation:

// Node.js relay — gpGet() implementation
function gpGet(hash) {
  const e = gpMempool.get(hash);
  if (!e) return null;

  clearTimeout(e.timer);           // cancel TTL timer
  const out = Buffer.from(e.blob); // copy for transmission
  e.blob.fill(0);                  // overwrite original in RAM
  gpMempool.delete(hash);          // remove from Map
  gpTotalOut++;

  return out;
  // Note: JavaScript GC will eventually collect the Buffer.from() copy
  // For guaranteed RAM wiping, use the native Rust relay implementation
}

Rust Crate

# Cargo.toml
[dependencies]
paramant-ghost-pipe = "2.0"
use paramant_ghost_pipe::{GhostPipe, Config};

async fn main() -> anyhow::Result<()> {
    let gp = GhostPipe::new(Config {
        relay: "https://relay.paramant.app".into(),
        api_key: Some("pgp_your_key".into()),
    });

    // Keygen — ephemeral ML-KEM-768 + ECDH P-256 hybrid
    let (pub_key, priv_key) = gp.keygen()?;

    // Send
    let result = gp.send(
        data: b"my secret payload",
        to: &pub_key_hex,
    ).await?;

    println!("Hash: {}", result.hash);
    println!("Expires: {}s", result.ttl_ms / 1000);

    // Receive (burn-on-read) — Zeroize on drop
    let plaintext = gp.receive(&result.hash, &priv_key).await?;
    // plaintext is ZeroizeOnDrop — wiped when it goes out of scope

    Ok(())
}
