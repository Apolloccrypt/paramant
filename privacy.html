// KERN
PARAMANT verzamelt geen persoonsgegevens.
Geen account. Geen e-mail. Geen telefoonnummer. Geen IP-opslag. Geen analytics.
De relay verwerkt versleutelde berichten maar slaat ze niet op. Na 5 minuten worden ze automatisch verwijderd.

// WAT WE NIET DOEN

PARAMANT gebruikt geen cookies, geen tracking pixels, geen fingerprinting en geen analytics.
Er zijn geen advertenties. Er zijn geen investeerders. Er is geen bedrijf achter PARAMANT.
De broncode is volledig open-source en verifieerbaar op GitHub.

// RELAY SERVER

De relay draait op een Hetzner server in Neurenberg, Duitsland (EU/GDPR-jurisdictie).
De relay ontvangt versleutelde pakketjes. De relay kan de inhoud nooit lezen — berichten zijn end-to-end versleuteld met ML-KEM-768 + AES-256-GCM voordat ze de relay bereiken.
Pakketjes worden na maximaal 5 minuten automatisch verwijderd.

Een Tor hidden service is beschikbaar voor gebruikers die ook de relay-connectie willen anonimiseren.
Via Tor ziet de relay het IP-adres niet.

// EXTERNE DIENSTEN

De website laadt momenteel twee externe bibliotheken via Cloudflare CDN (TailwindCSS, QRCode.js).
Cloudflare verwerkt hierbij uw IP-adres als onderdeel van de CDN-levering. Dit wordt niet door PARAMANT gecontroleerd.
Wij werken aan het volledig inline bundelen van deze bibliotheken om dit te elimineren.

// AVG / GDPR

Omdat PARAMANT geen persoonsgegevens verwerkt, is er formeel geen verwerkingsgrondslag vereist.
Er is geen Data Protection Officer aangesteld, geen verwerkingsregister en geen cookiemelding — omdat er niets te melden valt.
Betreffende GDPR artikel 5(1)(e): gegevens worden niet langer bewaard dan noodzakelijk. Relay: 5 minuten. Lokaal: op verzoek verwijderbaar.

// KINDEREN

PARAMANT is niet specifiek gericht op kinderen onder de 13 jaar. Er worden geen gegevens van minderjarigen bewust verzameld.
